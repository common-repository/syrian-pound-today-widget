A widget for displaying the currencies exchange rates versus the Syrian Pound

== Description ==

تستطيع الآن تنصيب هذه الإضافة التي تتيح لك إضافة قائمة جانبية لأسعار العملات مقابل الليرة السورية على موقعك والتي تتمتع بالتحديث التلقائي من موقع العملات الأول في سورية، موقع الليرة اليوم https://sp-today.com

When you install this plugin you will be able to put a widget in your sidebar that shows the currencies exchange rates versus the Syrian Pound, which is updated automatically from the first website in Syria for currencies exchange rates (Syrian Pound Today: https://sp-today.com )
